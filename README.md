# Bike Buyers Excel Analysis Project

## Overview
This repository contains an Excel-based data analysis project on bike purchase data. It includes data cleaning, pivot tables, charts, and an interactive dashboard to analyze customer demographics and buying behavior.

## Files
- **Excel Project Dataset.xlsx**: The main Excel file with:
  - **bike_buyers**: Raw data.
  - **working sheet**: Cleaned data with added age brackets.
  - **Pivot table**: Pivot calculations for income, commute, and age analyses.
  - **Dashboard**: Interactive dashboard with charts and slicers.

## Project Steps
1. **Data Cleaning**:
   - Removed duplicates.
   - Standardized values (e.g., gender, marital status).
   - Added age bracket column using IF formulas.

2. **Analysis with Pivots and Charts**:
   - Average income by gender and purchase.
   - Commute distance distribution by buyers.
   - Bike purchases by age brackets.

3. **Dashboard**:
   - Visual layout of charts.
   - Slicers for Marital Status, Education, Region, and Cars Owned for interactivity.

## Tools Used
- Microsoft Excel (formulas, pivot tables, charts, slicers).

## How to Use
- Open the Excel file.
- Go to the "Dashboard" sheet.
- Use slicers to filter and explore data.
